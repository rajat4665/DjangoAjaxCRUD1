# DjangoAjaxCRUD1
In this repository Ajax curd rule example has been shown 

<h3>How to run this app:</h3>
<ul>
	<li>clone this repository</li>
	<li>Install requirement file</li>
	<li>open directory where manage.py file is present</li>
	<li>open terminal  type the following command
<pre>pip3 install -r requirement.txt # this command install all requirements 
python3 manage.py createsuperuser  # this command will create super user for your admin panel
python3 manage.py runserver</pre>
